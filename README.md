# Linear Regression From Scratch

## Project Overview

This project implements Linear Regression from scratch using NumPy without relying on Scikit-Learn's LinearRegression model. The project predicts house prices and manually calculates evaluation metrics to understand the mathematical foundations of machine learning.

## Key Features

* Manual implementation of Linear Regression
* Normal Equation for parameter estimation
* Custom prediction function
* Manual calculation of:

  * R² Score
  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * Root Mean Squared Error (RMSE)
* House Price Prediction

## Technologies Used

* Python
* NumPy
* Pandas
* Jupyter Notebook

## Learning Outcomes

This project demonstrates an understanding of the mathematical foundations of Linear Regression and model evaluation beyond the use of high-level machine learning libraries.
